<h1>Books applicaton, back-end</h1>
<h3>Usage</h3>
<ul>
    <li>$ git clone https://github.com/AlexPurhalo/books-back.git</li>
    <li>$ cd books-back</li>
    <li>$ rbenv shell 2.3.1</li>
    <li>$ bundle install</li>
    <li>$ foreman start</li>
</ul>
